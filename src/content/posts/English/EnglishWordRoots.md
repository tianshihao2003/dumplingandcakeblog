# 🔓Day01

## **词根**：`radi`

**核心含义**：根（联想“抓地”的谐音记忆法）

**引申义**：由“根”引申为**光线、辐射**（如同树根向四周延伸）  
**词缀辅助**：  
后缀（如 `-ate`​ 表动词，`-ant`​ 表形容词，`-ion/-um` 表名词）

|单词|词性|中文释义|拆解与记忆逻辑|
| :-----| :-----| :-------------------| :---------------------------------------------------------------------------|
|**radical**|adj.|根本的；彻底的|​`radi`​ (根) + `-cal` (形容词后缀) → 从“根”上的 → 根本的、彻底的|
|**eradicate**|v.|根除|​`e-`​ (前缀：出/除去) + `radi`​ (根) + `-ate` (动词后缀) → 把根拔出来 → 根除、消灭|
|**radiate**|v.|辐射；发散；使显现|​`radi`​ (根/光线) + `-ate` (动词后缀) → 像根/光线一样向外扩散 → 辐射、散发|
|**radiant**|adj.|辐射的；容光焕发的|​`radi`​ (光线) + `-ant` (形容词后缀) → 向外散发着光的 → 容光焕发的、灿烂的|
|**radio**|n.|收音机|​`radi`​ (辐射/无线电波) + `-o` (名词后缀) → 利用无线电波（辐射）传播声音的设备|
|**radioactive**|adj.|放射性的|​`radio`​ (辐射) + `active` (活跃的) → 具有活跃辐射特性的 → 放射性的|
|**radium**|n.|镭|​`radi`​ (辐射) + `-um` (化学元素名词后缀) → 一种具有强放射性的化学元素|

# 🔓Day02

## **词根**：`pose`

**核心含义**：摆放（联想“摆pose”的日常场景）

**引申义**：由“摆放”引申为**放置、姿态、构成、提出**等与“放置/安排”相关的含义  
**词缀辅助**：结合之前学过的：  
前缀（如 `ex-`​ 向外、`com-`​ 共同、`pro-`​ 向前、`dis-`​ 分开、`op-`​ 相反、`de-`​ 向下/除去）  
后缀（如 `-it`​, `-ition`​, `-ent` 等）

|单词|词性|中文释义|拆解与记忆逻辑|
| :-----| :-----| :-------------------| :---------------------------------------------------------------------------|
|**expose**|v.|暴露；揭露；使显露|​`ex-`​ (向外) + `pose` (摆放) → 把东西摆到外面 → 暴露、揭露|
|**compose**|v.|组成；创造；创作|​`com-`​ (共同) + `pose` (摆放) → 把各部分摆在一起 → 组成、创作（如作曲/作文）|
|**composition**|n.|作文；作品；作曲|​`compos(e)`​ (组成/创作) + `-ition` (名词后缀) → 创作的成果 → 作文、作品|
|**composite**|adj.|复合的；合成的|​`compos(e)`​ (组成) + `-ite` (形容词/名词后缀) → 由多部分组成的 → 复合的|
|**propose**|v.|求婚；提名；提议|​`pro-`​ (向前) + `pose` (摆放) → 把想法/计划摆到前面 → 提议、求婚|
|**dispose**|v.|扔掉；处理；安排|​`dis-`​ (分开/离开) + `pose` (摆放) → 把东西摆到别处/移走 → 处理、扔掉|
|**oppose**|v.|反对|​`op-`​ (相反) + `pose` (摆放) → 摆到对面/对立位置 → 反对|
|**opponent**|n.|敌人；对手|​`oppos(e)`​ (反对) + `-ent` (表人的名词后缀) → 反对者 → 对手、敌人|
|**depose**|v.|废除；罢免|​`de-`​ (向下/除去) + `pose` (摆放) → 把某人从位置上放下来 → 罢免、废除|
|**deposit**|n.|沉淀物；定金|​`de-`​ (向下) + `pos(e)`​ (摆放) + `-it` → 向下放置的东西 → 沉淀物；存放的钱（定金）|

# 🔓Day03

## **词根**：`vers(t)`

**核心含义**：转（联想“转动的快速声响”）

**引申义**：由“转”引申为**转变、转移、反转、转向**等与“方向/状态改变”相关的含义  
**词缀辅助**：前缀（如 `re-`​ 向后、`a-`​ 离开、`con-`​ 共同、`di-`​ 分开、`intro-`​ 向内、`extro-`​ 向外、`ann(i)-`​ 年、`contro-` 相反）

|单词|词性|中文释义|拆解与记忆逻辑|
| :-----| :--------| :---------------------| :--------------------------------------------------------------------|
|**reverse**|v.|扭转；颠倒|​`re-`​ (向后/相反) + `verse` (转) → 向反方向转 → 扭转、颠倒|
|**avert**|v.|转移（注意力）；避免|​`a-`​ (离开) + `vert` (转) → 把目光/危险转开 → 转移、避免|
|**convert**|v.|转变；换算；改变信仰|​`con-`​ (加强/共同) + `vert` (转) → 彻底转变形态/立场 → 转变、改信|
|**divert**|v.|转移；改道（注意力）|​`di-`​ (分开/偏离) + `vert` (转) → 转离原方向 → 转移、改道|
|**introvert**|n./adj.|内向的人；内向的|​`intro-`​ (向内) + `vert` (转) → 性格转向内心 → 内向的人/性格|
|**extrovert**|n./adj.|外向的人；外向的|​`extro-`​ (向外) + `vert` (转) → 性格转向外部 → 外向的人/性格|
|**anniversary**|n.|周年纪念|​`anni-`​ (年) + `vers`​ (转) + `-ary` (名词后缀) → 时间转了一年 → 周年纪念|
|**controversy**|n.|争论；争吵；争议|​`contro-`​ (相反) + `vers`​ (转) + `-y` (名词后缀) → 观点转向对立 → 争论、争议|
